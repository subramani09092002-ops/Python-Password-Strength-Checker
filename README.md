# Python-Password-Strength-Checker


🔐 Password Strength Checker is a Python-based cybersecurity mini project that evaluates the strength of user passwords based on important security criteria such as minimum length, uppercase and lowercase letters, numbers, and special characters. The program provides real-time feedback, identifies missing security requirements, and classifies passwords as Weak, Medium, or Strong. This project was built to practice Python programming, input validation, conditional logic, and basic cybersecurity concepts.
